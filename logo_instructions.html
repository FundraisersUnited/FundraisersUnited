Logo Download Instructions

Step-by-step guide to get your logo from Looka.com

Method 1: Download Directly from Looka

  1. Visit your logo at Looka.com through this link: https://looka.com/editor/219944293
  2. Log in to your Looka account if prompted
  3. When your logo appears in the editor, you can:
    • Click on "Download" or "Export" buttons (usually in the top-right corner)
    • OR right-click on the logo and select "Save image as..."
  4. Save the file as fundraisers-united-logo.png in the images folder of your website project
Open Looka Editor

Method 2: Use a Screenshot Tool

  1. Open your logo in the Looka editor
  2. Take a screenshot of your logo:
    • Windows: Press Windows + Shift + S to use the Snipping Tool
    • Mac: Press Command + Shift + 4 to capture a selection
  3. Save the screenshot as fundraisers-united-logo.png in the images folder
Tip: Make sure to capture the logo with a transparent background if possible. If your logo has a colored background, try to isolate just the logo itself.

Method 3: Run Our Python Helper Script

  1. Make sure you have Python installed on your computer
  2. Open a command prompt or terminal in your website folder
  3. Run one of our helper scripts:
    • python logo_helper.py - Simple version with minimal dependencies
    • python download_logo.py - More advanced version (requires extra libraries)
  4. Follow the on-screen instructions in the tool
Note: The Python scripts may require additional libraries. If you encounter errors, please use Method 1 or Method 2 instead.

Where to Place Your Logo

After downloading the logo, make sure:

  1. The file is named fundraisers-united-logo.png
  2. It's saved in the images folder of your website project

Your website is already configured to use the logo in the header and footer once it's correctly placed in the images folder.
