# Fundraisers United

A web application for managing fundraising campaigns and donations.

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```
   git clone <repository-url>
   cd FundraisersUnited
   ```

2. Install dependencies
   ```
   npm install
   # or
   yarn install
   ```

3. Start the development server
   ```
   npm run dev
   # or
   yarn dev
   ```

## Features

- Create and manage fundraising campaigns
- Track donations
- User authentication
- Dashboard with analytics
